-   Hero landing section

## Getting Started

1.  Download or clone the project.
2.  Open the project folder.
3.  Add your images to the `images` folder.
4.  Replace the placeholder YouTube video IDs in `index.html`.
5.  Update your email address and social media links.
6.  Open `index.html` in a web browser.

## Customisation

### Artist Information

Update: - Artist name - Biography - Song titles - Contact details

### Images

Replace the placeholder images with your own.

### YouTube

Replace:

``` html
https://www.youtube.com/embed/YOUR_VIDEO_ID
```

with your own video IDs.
